# Tarefa_corre-o_tipagens_TS

Nesta atividade, foram corrigidas e ajustadas tipagens em TypeScript com o objetivo de garantir maior segurança, consistência e compatibilidade entre os dados utilizados na aplicação. Foram aplicados conceitos fundamentais da linguagem, como extensão de interfaces (extends), utilização do utilitário Omit, tipagem de funções, union types e definição correta de interfaces para objetos.

As correções asseguram que as tipagens estejam alinhadas com as validações propostas e com o uso real dos dados no código, evitando erros de compilação e fortalecendo as boas práticas de tipagem estática. Com isso, os requisitos da tarefa foram atendidos de forma objetiva e conforme solicitado no enunciado.
